# Kirill Glazunov – fertige Website

Dieser Ordner ist vollständig uploadfertig für das Repository `derhat88/ambien`.

Enthalten sind:

- `index.html` – die Website
- `styles.css` – das Dark-Mode / Liquid-Glass-Design

## Hochladen

1. Öffne https://github.com/derhat88/ambien
2. Wähle **Add file** → **Upload files**.
3. Ziehe nur `index.html` und `styles.css` in das Upload-Fenster.
4. Ersetze die vorhandenen Dateien und klicke auf **Commit changes**.

Die Datei `CNAME` bitte nicht hochladen, wenn du keine eigene Domain verwendest.

Danach aktualisiert GitHub Pages die Website automatisch. Das kann ein bis zwei Minuten dauern.

## GitHub Pages (nur falls die Seite noch nicht online ist)

Unter **Settings** → **Pages** als Quelle `Deploy from a branch`, Branch `main` und Ordner `/(root)` auswählen.
